# 業務週報メーカー

毎日の記録テキストから「業務週報(人事部用)」を作成し、A4縦1枚のPDFに出力する専用アプリです。
サーバー不要・単一HTMLファイルで、`index.html` をブラウザ(Chrome推奨)で開くだけで使えます。

## 使い方

1. `weekly-report/index.html` をダブルクリックしてブラウザで開く
2. ツールバーで **対象週(先週の月曜)** と **氏名** を設定(次回以降は自動保存されます)
3. **「📋 記録テキストから作成」** を開き、毎日の記録を貼り付けて **「解析して反映」**
   - 「月」「火」…と曜日だけの行で日を区切った形式(`7/7(月)` の形式もOK)
   - 全角スペース区切りは自動で「 / 」に変換されます(チェックで無効化可)
   - 勤務時間(例 `9-18`)から「時間」を自動計算(7時間以上は休憩1時間を控除)
4. シート上の各セルは **直接クリックして編集** できます
   - 成果①〜⑤ / 観察a〜e / 分析a〜e / その他 / 今週行動予定 / フリーライティング
   - 貼り付けたテキストの一部を選択して「→ 成果へ」等のボタンで転記も可能
5. **「🖨 PDFに保存(印刷)」** → 印刷ダイアログで「PDFに保存」を選択 → A4縦1枚のPDFが完成

## 機能

- **自動保存**: 入力内容はブラウザ(localStorage)に自動保存。閉じても消えません
- **書き出し/読み込み**: JSONファイルとして保存・復元(週ごとのアーカイブに)
- **1枚判定**: 内容が多くA4 1枚に収まらない恐れがある場合は警告を表示
- **合計時間**: 「時間」列の合計を自動計算
- **URL共有**: `#d=<base64>` ハッシュで入力済み状態のURLを生成可能(自動生成用)

## 記録テキストの書き方(例)

```
月
新入社員　入社式　予行演習実施のあと本番　滞りなく
書類記入　オリエンテーション
火
引継ぎ詳細
クルー採用の打ち手まとめ
```

「サンプル」ボタンで記入例を確認できます。
