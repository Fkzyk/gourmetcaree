# 全クルーアンケート提出確認ツール（GAS）

「26.4 学生クルー対象アンケート（回答）」の提出確認ツールと同じ仕様で、
「【2026】全クルー対象アンケート（回答）」向けにカスタマイズした Google Apps Script です。

## できること

| 機能 | 内容 |
| --- | --- |
| リスト更新（未提出者抽出） | 従業員リストから対象者（パート・アルバイト / 在籍中 / 店舗所属）を抽出し、フォーム回答と照合。「未提出者」「回答率_店舗別」「回答率_営業部別」「外国人リスト」を自動生成 |
| 進捗確認シート更新 | 回答を「店舗名 / 従業員番号 / 氏名 / 進捗状況 / 架電 / …」の並びで転記。進捗状況プルダウン（SMS送信済〜対象外の7段階）と架電チェックボックスを自動設定。重複回答は最新のみ。手入力済みの進捗は保持 |
| 共有用ファイル出力 | 未提出者・回答率シートだけを別スプレッドシートに書き出す（値のみ）。従業員リスト等の個人情報を見せずに集計結果を共有できる。2回目以降は同じファイルを上書きするので共有リンクは不変 |
| 初期セットアップ | 不足シート（除外リスト・従業員リスト・店長・営業部長・テンプレ・操作マニュアルなど）の雛形を自動作成 |

> **シートの閲覧制限について**: Googleスプレッドシートに「特定シートのパスワード保護」機能はありません。
> シートの保護は編集防止のみで閲覧は防げず、非表示シートも閲覧権限があれば
> `IMPORTRANGE` やファイルコピーで見えてしまいます。見せたくないシートがある場合は
> 「共有用ファイル出力」で見せたいシートだけの別ファイルを共有してください。

## 導入手順

1. **xlsx を Google スプレッドシートに変換する**
   「調整するやつ【2026】全クルー対象アンケート（回答）.xlsx」は Excel 形式のため、
   そのままでは GAS を追加できません。ファイルを開き
   `ファイル > Google スプレッドシートとして保存` で変換してください。
   ※ フォームと連携する場合は、フォームの回答先をこのスプレッドシートに設定してください。
2. 変換したスプレッドシートで `拡張機能 > Apps Script` を開く
3. `Code.gs` の内容をすべて貼り付けて保存
4. エディタ上で `setupSheets` を一度実行（初回は権限の承認が必要）
5. スプレッドシートに戻り、以下のマスターデータを貼り付ける
   - **従業員リスト** … 給与システム出力の全スタッフデータ（学生版のシートをコピーでも可）
   - **店長・営業部長** … 店舗と営業部の紐付けマスター（学生版のシートをコピーでも可）
   - **除外リスト** … 対象外にしたい人の社員 No（任意）
6. メニュー `📝 全クルーアンケート > すべて更新` を実行

## 学生版との違い

- 対象は **22歳以上** のパート・アルバイト。年齢条件を変える場合は
  `Code.gs` 冒頭の `CONFIG.AGE_MIN` / `CONFIG.AGE_MAX` を変更（`null` で制限なし）
- フォームの設問（①店舗名〜⑨キャリアアップ意向）に合わせて列を自動判定
- 電話番号はフォームで直接回収されるため、進捗確認シートに数字のみに正規化して転記
